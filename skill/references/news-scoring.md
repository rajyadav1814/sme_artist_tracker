# News Desk — Scoring & Editorial Reference

## Newsworthiness Scoring Matrix

Each detected change is scored 0–10. The top 15 highest-scoring items form the daily briefing.
Ties broken by: (1) artist reach tier, (2) recency, (3) data confidence.

### Scoring Rules

| Signal Category                  | Base Score | Multipliers                                        |
|----------------------------------|------------|-----------------------------------------------------|
| **Milestone crossing**           | 10         | ×1.5 if round number (10M, 50M, 100M)              |
| **New release (single/album)**   | 9          | ×1.3 if collab with another roster artist           |
| **Chart entry / movement**       | 9          | ×1.5 if #1 position; ×1.2 if Billboard             |
| **Award / nomination**           | 8          | ×1.5 if Grammy/Latin Grammy                         |
| **Viral spike**                  | 8          | ×1.2 per order of magnitude above average           |
| **Collaboration detected**       | 8          | ×1.2 if cross-genre; ×1.3 if inter-roster           |
| **Rapid follower surge**         | 7          | ×1.1 per % point above 2% threshold                |
| **Engagement anomaly**           | 7          | ×1.2 if sustained >24 hours                         |
| **Controversy / PR event**       | 7          | ×1.3 if mainstream press coverage                   |
| **Tour / event announcement**    | 6          | ×1.2 if international dates                         |
| **Platform silence**             | 5          | ×1.3 if previously high-frequency poster            |
| **Sustained metric decline**     | 5          | ×1.1 per consecutive day of decline                 |

### Editorial Templates

**Milestone:**
> {ARTIST} has crossed {NUMBER} followers on {PLATFORM}, adding {DELTA} in the past
> {TIMEFRAME}. This places them {CONTEXT — e.g., "among the top 5 most-followed Latin
> artists on the platform"}.

**New Release:**
> {ARTIST} dropped "{TITLE}" {TODAY/YESTERDAY}, their first release in {DAYS} days.
> Early signals show {METRIC — e.g., "2.1M Spotify streams in the first 12 hours"}.

**Viral Spike:**
> A {PLATFORM} post by {ARTIST} is outperforming their average by {MULTIPLIER}×, pulling
> {ENGAGEMENT} interactions in {TIMEFRAME}. The post {BRIEF DESCRIPTION}.

**Collaboration:**
> {ARTIST_1} and {ARTIST_2} appear to be teaming up — {EVIDENCE}. This marks {CONTEXT}.

**Anomaly (Positive):**
> {ARTIST}'s engagement rate jumped to {RATE}%, up from a {TIMEFRAME} average of {AVG}%.
> The surge coincides with {LIKELY CAUSE}.

**Anomaly (Negative):**
> {ARTIST} has lost {NUMBER} {FOLLOWERS/LISTENERS} on {PLATFORM} over the past {DAYS} days,
> a {PERCENT}% decline. {CONTEXT — e.g., "This follows a period of inactivity on the platform"}.

**Platform Silence:**
> {ARTIST} has not posted on {PLATFORM} in {DAYS} days, breaking a pattern of {FREQUENCY}
> posting. Their last post on {DATE} {BRIEF DESCRIPTION}.

## Data Confidence Indicators

Each news item carries a confidence badge:

- ●●●●● **Verified**: Direct from platform, fetched today
- ●●●●○ **Recent**: From aggregator or search, <48 hours old
- ●●●○○ **Estimated**: Multiple sources averaged, <7 days old
- ●●○○○ **Stale**: Best available data, >7 days old
- ●○○○○ **Inferred**: Derived from indirect signals (e.g., chart position implying streams)
